# Agente de IA - Mercado Central 24 Hs 🚀

Este proyecto es un agente conversacional inteligente diseñado para resolver consultas de inventario, políticas de compra, atención al cliente (ATC) y reglamentos internos para **Mercado Central 24 Hs**.

## 🛠️ Características del Proyecto
* **Historial de evolución completo:** El repositorio cuenta con una reconstrucción cronológica de las 27 etapas del desarrollo de la aplicación.
* **Procesamiento de Lenguaje Natural (NLP):** Utiliza modelos locales de embeddings semánticos para entender las preguntas de los usuarios y buscar en los manuales de la empresa.
* **Panel Interactivo:** Diseñado con una interfaz amigable usando Gradio.

## 📦 Tecnologías Utilizadas
* Python 3
* Gradio
* Sentence-Transformers
* PyPDF
* Pandas & Openpyxl
